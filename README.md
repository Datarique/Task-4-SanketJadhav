# DecodeLabs Project 4 - Data Visualization

## Tools Used
- Python
- Pandas
- Matplotlib

## Charts Created
1. Total Sales by Product
2. Payment Method Distribution
3. Monthly Sales Trend
4. Quantity vs Total Price

## Insights
- Product sales vary across categories.
- Online and card payments are commonly used.
- Sales fluctuate across months.
- Higher quantities generally result in higher total sales.
